Ethiopic in GUS

Status

Complete. No outstanding questions remain for Ethiopic.

IPA mappings for Ethiopic script are language and cultually sensitive. For characters that go unused in the orthography of a given language the default phonemic value in the applicable language for the character is applied. For example U+1278 was not used in Ge'ez language nor orthography so U+02A7 of Amharic and Tigrigna is applied in the Ge'ez mappings.
